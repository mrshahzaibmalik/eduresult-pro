Monthly Working Days

Set the number of working days for each month. These are used to calculate Attendance % and Days Absent in the Monthly Attendance section. Formula: Att% = Present ÷ Working Days × 100

Attendance Status Rules

These thresholds determine the status label shown on attendance reports.

Changes apply after clicking "Save Settings"
Subject Configuration

Configure subject names and maximum marks. Monthly Max sets each subject's max marks for monthly tests. Set Final Max to 0 if subject is Mid Term only.

Grade Boundaries

Set minimum percentage for each grade.

Performance Categories

These are the behavioural rating categories shown on result cards.

Student Info Fields

Define extra fields for every student — Father's Name, Teacher Name, Phone, Address, or anything you need. Toggle Show in Sheet to display a column in the Mid / Final / Monthly marks tables.

📋 How to Give Login Credentials to Users

This system uses admin-assigned credentials only. Users cannot register themselves — the Admin creates every account and personally hands the username & password to each person.

👑
Shahzaib — Permanent Super Admin
The Shahzaib account is a permanent, undeletable Super Admin. Only Shahzaib can delete other accounts. No one can remove or demote Shahzaib's account. Login: shahzaib / shahzaib123
1 Click "Add Account"

Press the + Add Account button (top-right of this page) to open the account creation form.

2 Fill in the Details

Enter the person's Full Name, choose a Username, set a Password, select their Role (Teacher / Coordinator / Admin), and optionally assign Classes so they see only those classes.

3 Share Credentials

After saving, tell the user their username & password directly — in person, via WhatsApp, or written on paper. The system does not send emails.

4 Change Password Anytime

Click the key icon next to any account to edit the name, username, password, or role at any time.
